# Example Blog Post

This is an example blog post written in Markdown. You can use all standard Markdown features here.

## Writing Your Posts

To add a new blog post:

1. Create a new `.md` file in the `posts/` folder
2. Write your post in Markdown
3. Add an entry to the `blogPosts` array in `js/blog.js` with:
   - `id`: A unique identifier
   - `title`: The post title
   - `date`: Publication date (YYYY-MM-DD format)
   - `filename`: The name of your markdown file

## Markdown Features

You can use **bold text**, *italic text*, and [links](https://example.com).
